# 基本操作

## 打开终端

ubuntu

```
ctrl + alt + T
```

## Linux 删除（清空）已输入的命令

有时输入了很长的 Linux 命令之后发现不想执行这个命令，按 backspace 键删除又要等，其实可以使用如下快捷键

删掉（清空）已输入的命令，不让其执行：

`ctrl` + `u`

其他快捷键：

跳到命令行行头：`ctrl` + `a`

跳到命令行结尾：`ctrl` + `e`

## 终端内 打开新标签页

`ctrl` + `shift` + `T`

## 命令行中 查看已安装的软件包

```bash
apt list --installed
```

## apt remove 和 apt purge 的区别

apt remove 只是删除包的二进制文件。它留下了残留的配置文件。

apt purge 删除与包相关的所有内容，包括配置文件。

## 查看端口占用

```bash
ss -tuln
```

## 24.04 开启 SSH 远程登陆

安装 `openssh-server`

```bash
sudo apt install openssh-server
```

安装完成后查看 `服务` 状态

```bash
systemctl status ssh
```

如果服务未启动则重启 `服务`

```bash
systemctl restart ssh
```

将 `服务` 添加到开机启动

```bash
systemctl enable ssh
```

## 24.04 安装 tmux 分屏终端

```bash
sudo apt install tmux
```

新建会话

```bash
tmux new -s my_session
```

新建会话后直接输入 `tmux split-window` 划分窗格

划分为上下两个

```bash
tmux split-window
```

划分为左右两个

```bash
tmux split-window -h
```

## tmux 窗格快捷键

划分左右两个窗格

- 先按下组合键 `Ctrl` + `b` 再按下 `%`

划分上下两个窗格

- 先按下组合键 `Ctrl` + `b` 再按下 `"`

光标切换到其他窗格

- 先按下组合键 `Ctrl` + `b` 再按下 `方向键`

光标切换到上一个窗格

- 先按下组合键 `Ctrl` + `b` 再按下 `;`

光标切换到下一个窗格

- 先按下组合键 `Ctrl` + `b` 再按下 `o`

当前窗格与上一个窗格交换位置

- 先按下组合键 `Ctrl` + `b` 再按下 `{`

当前窗格与下一个窗格交换位置

- 先按下组合键 `Ctrl` + `b` 再按下 `}`

所有窗格向前移动一个位置，第一个窗格变成最后一个窗格

- 先按下组合键 `Ctrl` + `b` 再按下 `Ctrl` + `o`

所有窗格向后移动一个位置，最后一个窗格变成第一个窗格

- 先按下组合键 `Ctrl` + `b` 再按下 `Alt` + `o`

关闭当前窗格

- 先按下组合键 `Ctrl` + `b` 再按下 `x`

将当前窗格拆分为一个独立窗口

- 先按下组合键 `Ctrl` + `b` 再按下 `!`

当前窗格全屏显示，再使用一次会变回原来大小

- 先按下组合键 `Ctrl` + `b` 再按下 `z`

按箭头方向调整窗格大小

- 先按下组合键 `Ctrl` + `b` 再按下 `Ctrl` + `方向键`

显示窗格编号

- 先按下组合键 `Ctrl` + `b` 再按下 `q`

## 24.04 安装 tree 显示文件夹结构

```bash
sudo apt install tree
```

-d 只显示目录名
-D 列出文件或目录的更改时间
-f 显示完整的相对路径名称
-t 按文件和目录的更改时间排序
-s 列出文件或目录大小
-u 列出文件或目录的拥有者名称，没有对应的名称时，则显示用户识别码
